## 🛠 Installation Instructions

1. **Download** the file `4chan.theme.css` from this repository.
2. Open Discord and navigate to **Vencord Settings** > **Themes**.
3. Click the **Open Themes Folder** button.
4. **Paste** the downloaded file into the folder that opens.
5. Back in Discord, click the **Load Missing Themes** button.
6. Locate the **4chan theme** in your list and toggle it to **Enable**.
